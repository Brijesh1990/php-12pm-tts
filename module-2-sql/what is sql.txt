what is sql ?
sql stands for structured query language 
sql is used to create a datatabase and tables structures 
sql is a case insenstive language
ex:INSERT or Insert or insert 
sql is provides some query or command to create table structures...

types of query or commands in sql ...

a) DDL 
b) DML
c) DQL
c) TCL 


a) DDL : ddl stands for data definition language 
       Note : ddl is used to create a database and tables structures 

      DDL query : create | alter | truncate | drop | rename | change 


    1) create database : create is used to create database and tables structured 

      syntax : create database databasename;

      examples : create database flip_db; 


    2) create table : create table is used to create a table structured
              table contains row and column    

      syntax : create table tablename 
      (
         columnname data-type(size) primary key auto_increment,

         columnname data-type(size) 


      )   


      examples : create table tbl_register(

    rid int PRIMARY key AUTO_INCREMENT,
    name varchar(255),
    email varchar(255),
    password varchar(255),
    phone bigint,
    address text

)


   examples : create table tbl_contact(

    contactid int PRIMARY key AUTO_INCREMENT,
    name varchar(255),

    phone bigint,

6: change : change is used to change a column name is a tables.

        examples : alter table tbl_contactus CHANGE  comment message text;

     d) DQL : stands for data query language

               what is order by and group by ..........

               a) Order by : order by is used to filter data from tables in asc and desc orders 

                  ex: select * from tbl_employee order by empid asc;
                      select * from tbl_employee order by name asc;
                      select * from tbl_employee order by name desc;


               b) group by : group by is used to filter data from tables in group of columns 

                    ex: select sum(salary),department from tbl_employee group by department;    


               7) particulars column name select 

                   ex: select empid,name,age from tbl_employee;  


               8) searching with like oeprator 

                   1) search name from tables name start with 'a' letter

                        ex:  select * from tbl_employee where name like 'a%';


                   2) search name from tables name end with 'h' letter

                        ex:  select * from tbl_employee where name like '%h';

                   3) search name from tables name start or end with 'a' 

                    select * from tbl_employee where name like '%a%';



               SQL function :

                  sql provides two types of function ....
                  
                 a) aggrigate function
                   
                  examples : avg() | sum() | max() | min() | count()

                  1) avg () : find average 

                           select avg(salary) as average_salary from tbl_employee;

                  2) sum() : sum of salary

                           select sum(salary) as sum_salary from tbl_employee;

                  3) max() : max of salary

                           select max(salary) as max_salary from tbl_employee;

                  4) min() : min of salary

                           select min(salary) as min_salary from tbl_employee;

                  5) count() :count a numbers 

                             select count(empid) as total_employee from tbl_employee;

                         

     
                 b) scalar function 

                  examples : first() | last()  | lcase() | ucase() 

                     a) first() is used to fetch first rows data only

                         ex: select first(name) from tbl_employee;


                     b) last() is used to fetch last rows data only 
               
                         ex: select last(name) from tbl_employee;

                     c) lcase is used to convert an rows in lower case
                         ex: select lcase(name) from tbl_employee;

                     d) ucase is used to convert an rows in uppercase case    

                          ex: select ucase(name) from tbl_employee;   





                 subquery in sql : query within another query 

                    examples : select max(salary) as second_highest_salary from tbl_employee where salary < (select max(salary) from tbl_employee);
